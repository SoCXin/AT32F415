## [描述](https://github.com/SoCXin/MM32SPIN120/wiki) 

[MM32SPIN120](https://github.com/SoCXin/MM32SPIN120) 是电机专用类SoC，产品内嵌1个比较器

集成了2个GateDriver，支持外挂功率 N 型 MOSFET 驱动器，工作电压4.5V ∼ 15V。GateDriver还支持电压UVLO保护。
